NexCorp Business Corporate Template
===================================
Version: 1.0.0
Author: Google Deepmind Agent
Date: 2026-01-31

Description
-----------
A professional, responsive, and clean HTML5/CSS3 template designed for corporate businesses, agencies, and consultancies. Built with semantic HTML, modern CSS variables, and Vanilla JavaScript.

Features
--------
- 7 Pre-designed HTML Pages
- Fully Responsive (Mobile-First)
- Clean & Semantic Code
- SEO Optimized
- Fast Performance (Lazy Loading integrated)
- Easy to Customize (CSS Variables)

Folder Structure
----------------
/
├── index.html          # Homepage
├── about.html          # About Us
├── services.html       # Services
├── projects.html       # Portfolio/Projects
├── pricing.html        # Pricing Plans
├── blog.html           # Blog Listing
├── contact.html        # Contact Page
│
└── assets/
    ├── css/            # CSS Files
    │   ├── style.css       # Global Styles
    │   ├── index.css       # Home Page Styles
    │   ├── about.css       # Page Specific Styles...
    │   └── ...
    │
    └── js/             # JavaScript Files
        ├── main.js         # Global Scripts
        └── ...


Installation & Usage
--------------------
1. Unzip the package.
2. Open any .html file in your web browser to view the template.
3. Open the files in your code editor to start customizing.
4. Refer to the 'Documentation' folder for detailed instructions.
